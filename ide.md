# eclipse快捷键

- <leader>v c-v块模式
- <leader>w c/c++搜索关键字，主要通过快捷键导出的excel表查看快捷键

# vs

- 右键 - a 设置启动项
- 右键 - x 打开文件目录
- 调试相关
  - 打开测试窗口：Alt + S
  - 运行所有测试：Ctrl + R，A
  - 重复上次运行：Ctrl + R，L
  - 调试所有测试：Ctrl + R，Ctrl + A
  - 调试上次运行：Ctrl + R，D
  - 附加到进程 ：Ctrl + Alt + P
  - 调试单步执行 ：F10
  - 开始调试 ：F5 　
  - 停止调试：Shift + F5 　　
  - 添加快捷匹配: Ctrl + Alt + Q
  - 设置或删除断点: F9
- 生成相关
  - 生成解决方案：Ctrl + Shift + B
  - 对解决方案运行代码进行分析：Alt + F11
  - 生成项目：Ctrl + B
  - 编译：Ctrl + F7
